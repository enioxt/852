O sistema 852 já tem um motor de agregação AI Reports v1 que analisa 20 conversas últimas + 20 relatos para gerar insights cruzados. Queremos melhorar isso em **Phase 1** adicionando:

### 3️⃣ **Fuzzy Semantic Clustering** (Week 2, 4-5 hours)

**What:** Group similar insights automatically (e.g., "falta viatura" + "sem carro" = duplicates)

**Implementation:**
- Use `fuse.js` (fuzzy search library, already in `package.json`)
- Threshold: 0.7 similarity → merge insights
- Merge logic: Keep highest-scored insight, aggregate vote counts

**Where:**
- `/home/enio/852/src/lib/clustering.ts` (NEW FILE)
  - Export: `clusterInsights(insights, threshold=0.7)`
  - Logic: Levenshtein distance + keyword overlap

**Pipeline Integration:**
```typescript
// In report-generator.ts aggregateInsights():
const insights = await extractInsightsFromConversations(...);
const weighted = insights.map(i => ({ ...i, score: i.score * weights[i.category] }));
const clustered = clusterInsights(weighted, 0.7);  // ← NEW
const sorted = clustered.sort((a,b) => b.score - a.score);
```

**Expected Impact:** ⭐⭐⭐ (high) — 30-40% reduction in duplicate insights

**Files to Modify:**
1. `src/lib/clustering.ts` (NEW, +80 lines)
2. `src/lib/report-generator.ts` (integrate clustering, +5 lines)

### 4️⃣ **Knowledge Base Integration** (Week 2, 3-4 hours)

**What:** Weight insights by credibility of linked KB articles

**KB Exist:** `knowledge_base_852` table (created in migration 20260325000002)
- Fields: `id, title, topic, credibility_score (0-1), source`

**Implementation:**
- When insight matches KB topic → multiply score by credibility_score
- Example: "efetivo" insight + KB article "PCMG Staffing Policy (credibility: 0.95)" → boost score

**Where:**
- `/home/enio/852/src/lib/knowledge-weighting.ts` (NEW FILE)
  - Export: `applyKnowledgeWeighting(insights, kb_articles)`

**SQL Query (in supabase.ts):**
```sql
SELECT i.*, k.credibility_score
FROM ai_insights i
LEFT JOIN knowledge_base_852 k ON i.topic ~* k.topic
WHERE k.credibility_score IS NOT NULL
```

**Expected Impact:** ⭐⭐ (medium) — Better ranking of evidence-backed insights

**Files to Modify:**
1. `src/lib/knowledge-weighting.ts` (NEW, +60 lines)
2. `src/lib/report-generator.ts` (integrate KB weighting, +3 lines)
