# Sindpol-MG Inteligência 🚨

Plataforma anônima e segura baseada no ecossistema EGOS para coleta, estruturação e análise de relatos de Policiais Civis.

## Funcionalidades
- **Chatbot Institucional Seguro:** Focado em coletar fluxos de processos e problemas estruturais.
- **Agent IA (Qwen/GPT):** Treinado para esconder dados sensíveis (nomes, CPF, processos) e evitar duplicidade.
- **Exportação Fácil:** Arquivos `.md`, `.pdf`, `.docx` e link do WhatsApp integrados.
- **Dashboard de Insights:** Visualização em tempo real (Mockup) sobre gargalos institucionais.
- **Mobile First & Dark Mode:** Identidade visual inspirada na Inteligência Policial.

## Stack
- Next.js 16 (App Router)
- TailwindCSS 4 + Lucide Icons + Recharts
- Vercel AI SDK (@ai-sdk/react + @ai-sdk/openai)
- Bun

## Rodando o Projeto

1. Clone e entre no projeto.
2. Instale: `bun install`
3. Configure o `.env` seguindo `.env.example`
4. Rode: `bun run dev`
